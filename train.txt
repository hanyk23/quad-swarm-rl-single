改动要点：
1. 飞行速度从原来的高速配置收到 `quads_velocity_yaw_max_speed=1.0`。
2. 避障控制从固定反推改成基于实时速度的 PID：越靠近柱子、越高速朝柱子接近，反向速度越强。
3. 原始 checkpoint 已复制到 `train_dir_pid`，继续训练不会覆盖原版模型。


继续训练 PID 版：
cd /home/lzh/drone/quad-swarm-rl-single
bash train_snapshot_lidar_pid.sh

如果当前 shell 看不到 GPU，就用 CPU 验证：
cd /home/lzh/drone/quad-swarm-rl-single
DEVICE=cpu NUM_WORKERS=1 NUM_ENVS_PER_WORKER=2 bash train_snapshot_lidar_pid.sh

只训练到指定总步数，例如继续到 1400 万帧：
cd /home/lzh/drone/quad-swarm-rl-single
TRAIN_FOR_ENV_STEPS=14000000 bash train_snapshot_lidar_pid.sh


看 PID 版 latest：
cd /home/lzh/drone/quad-swarm-rl-single
bash enjoy_snapshot_lidar_pid.sh

默认仿真时长已经改成 5000 frame，也就是 50 秒。
如果想临时改成别的时长，例如 30 秒：
cd /home/lzh/drone/quad-swarm-rl-single
EPISODE_DURATION=30.0 bash enjoy_snapshot_lidar_pid.sh
